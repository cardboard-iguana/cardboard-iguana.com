# Using “xxd”

The xxd command is actually pretty standard on Linux systems.

```bash
xxd $BINARY $HEXDUMP    # Create a hex dump of binary file $BINARY.
xxd -r $HEXDUMP $BINARY # Reconstitute a binary file from a hex dump!
                        # Wow!
```
