# How to Set PATH in a Session

author:: Nathan Acks  
date:: 2022-04-25

Update the current PATH in Windows:

```powershell
set PATH=%PATH%;C:\Some\Other\Directory
```

Update the current PATH in Linux:

```bash
export PATH=$PATH:/some/other/directory
```
