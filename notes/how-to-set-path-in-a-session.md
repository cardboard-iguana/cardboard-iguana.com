# How to Set PATH in a Session

author:: Nathan Acks  
date:: 2022-04-25

Sometimes you want to update the PATH for your current shell, without making a permanent change.

## Windows

```powershell
set PATH=%PATH%;C:\Some\Other\Directory
```

## Linux

```bash
export PATH=$PATH:/some/other/directory
```
