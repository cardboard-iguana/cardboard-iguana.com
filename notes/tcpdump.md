# Using "tcpdump"

* **author**:: Nathan Acks
* **date**:: 2022-04-03

```bash
# Quick-n-dirty packet capture
#
sudo tcpdump port $PORT_TO_FILTER_ON -A
```

The `-A` flag prints packet contents in ASCII. Wireshark is obviously nicer.
