# Enumerate NFS Shares

## NMAP

NFS shares can be enumerated by [nmap](nmap.md) during scanning:

```bash
nmap -v -sT --script nfs-ls,nfs-statfs,nfs-showmount -p$PORT $IP
```

Normally `$PORT` is 111.
