# Linux Process Signals

author:: Nathan Acks  
date:: 2021-09-16

| Signal  | Description                                                        |
|:------- |:------------------------------------------------------------------ |
| SIGTERM | Kill the process, but allow it to do some cleanup tasks beforehand |
| SIGKILL | Kill the process - doesn't do any cleanup after the fact           |
| SIGSTOP | Stop/suspend a process                                             |
